# 🛒 EasyShop – Online Shopping Website

EasyShop is a modern and responsive e-commerce web application built using **HTML, Tailwind CSS, and JavaScript**. It allows users to browse products, search, filter by category and price, and manage a shopping cart.

---

##  🌐 Live Demo

EasyShop is live at - https://easyshop008.netlify.app/

## 🚀 Features

- 🔍 Product Search
- 🧩 Category Filtering (Electronics, Fashion, etc.)
- 💰 Price Filtering
- 🛒 Add to Cart functionality
- 📦 Dynamic product loading (FakeStore API)
- 📊 Cart item count update
- 🎨 Responsive UI using Tailwind CSS
- 🔔 Alerts using SweetAlert2
- 📌 Sticky Navbar + Search + Filters
- 📱 Mobile-friendly design

---

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS**
- **JavaScript (Vanilla JS)**
